## Exam at a glance

- Format: 2-hour **Moodle quiz** (text answers, simple calculations, multiple choice).
- Allowed: **2 A4 sheets (4 pages)** of personal notes + non-programmable calculator.
- High-yield, computable topics: precision/recall/F1 (macro & weighted), UAS/LAS, KL divergence, RLHF/DPO loss, MCC, Spearman.

## Notes are editable

Everything is Markdown + a LaTeX cheat sheet, so you can fork and adapt. The cheat sheet compiles with pdfLaTeX (locally or on Overleaf) — see [`Study/Cheatsheet/README_cheatsheet.md`](Study/Cheatsheet/README_cheatsheet.md).
